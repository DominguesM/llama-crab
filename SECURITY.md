# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

**Please do not open a public issue.**

Email `security@llama-crab.dev` (or open a private security advisory via the
GitHub Security tab) with a description of the vulnerability and reproduction
steps. We will respond within 72 hours.
