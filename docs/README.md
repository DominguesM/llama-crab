# docs/ — `llama-crab` user guide

This is the source for the **`llama-crab`** user guide, served via
[mdBook](https://rust-lang.github.io/mdBook/). Build it with:

```bash
cargo install mdbook
mdbook build docs/   # → docs/book/
mdbook serve docs/   # http://localhost:3000
```

Each chapter is one Markdown file in `docs/src/`. The table of contents
is `docs/src/SUMMARY.md`.

The crate's rustdoc is the authoritative API reference and is
auto-generated by `cargo doc`. This book is a higher-level narrative
that ties the public API to the underlying concepts.
