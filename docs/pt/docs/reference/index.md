# Referência

As páginas desta seção são a referência canônica para a API do
`llama-crab`. Use-as para procurar um tipo específico, uma feature
do Cargo, ou um nome de template de chat.

<div class="grid cards" markdown>

-   :material-folder-tree: __[Layout dos crates](crate-layout.md)__

    O workspace `llama-crab`: qual crate possui qual módulo, o que
    cada módulo faz, e onde adicionar novo código.

-   :material-package-variant: __[Features do Cargo](cargo-features.md)__

    A lista completa de features, com a descrição longa de cada
    uma.

-   :material-message-text-outline: __[Templates de chat embutidos](chat-templates.md)__

    Os 14 templates nomeados, as arquiteturas que cobrem, e um
    snippet de cada um.

-   :material-tag-outline: __[MSRV & versionamento](msrv.md)__

    A versão mínima suportada de Rust, a política de `SemVer` e o
    pin do commit do llama.cpp.

</div>
