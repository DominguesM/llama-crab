## Default Permission

Allows local LLM inference through llama-crab.

#### This default permission set includes the following:

- `allow-load-model`
- `allow-unload-model`
- `allow-list-models`
- `allow-retrieve-model`
- `allow-create-chat-completion`
- `allow-stream-chat-completion`
- `allow-create-completion`
- `allow-stream-completion`
- `allow-create-embedding`
- `allow-create-rerank`
- `allow-tokenize`
- `allow-tokenize-count`
- `allow-detokenize`
- `allow-cancel`
